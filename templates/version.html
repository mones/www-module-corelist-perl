Find out if a perl module is in core and since when.

You can enter the full name of the module. Search is case insensitive. If the module is not found, a search for the substring is started. If you choose substring search, a search for the substring is done in any case.
Examples: list::util, AutoLoader, perlio::scalar, Test (substring), test
